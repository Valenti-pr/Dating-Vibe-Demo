# PROJECT OVERVIEW — Онлайн-знакомства в 2026

## Суть проекта
Одностраничный showcase-сайт, демонстрирующий возможности вайбкодинга через тематику онлайн-знакомств 2026 года. Каждая секция — отдельная визуальная история, показывающая конкретную технику современной веб-разработки.

## Цели
- Вдохновить разработчиков и новичков на изучение vibe coding
- Показать, что можно создать за 1 день с Claude + Cursor
- Продемонстрировать современные тренды UI/UX 2025–2026
- Показать реальные возможности: анимации, интерактив, визуальные эффекты

## Целевая аудитория
**Первичная:** Разработчики и дизайнеры, интересующиеся AI-driven разработкой  
**Вторичная:** Люди, интересующиеся онлайн-знакомствами и цифровыми трендами  
**Ключевой инсайт:** Они хотят вдохновения, а не инструкций — сайт должен показывать, а не рассказывать

## Ключевые метрики успеха
- Визуальный wow-эффект с первых секунд
- Каждая секция удерживает внимание > 10 секунд
- Сайт грузится < 3 секунд на мобиле
- Lighthouse score > 85

## Тематика контента
Весь контент — про онлайн-знакомства 2026:
- AI-matching алгоритмы
- Дейтинг в VR/AR
- Профили нового поколения
- Тренды: голосовые профили, mood tags, vibe-compatibility
- Статистика рынка дейтинг-приложений
